# Minhero-like
I played a lot of Flash games in my youth, and decided to have another go for nostalgia's sake. When playing [Min Hero: Tower of Sages](https://www.kongregate.com/en/games/toychestgames/min-hero-tower-of-sages), I, in my infinite hubris, went and thought, "Yeah, I could make something like this.".

This is the result.
